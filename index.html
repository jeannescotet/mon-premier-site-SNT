Le HTML signifie Hyper Text Markup Language. Il désigne un type de language informatique déscriptif.
Ce language de balisage est utilisé pour la création de page web permettant notamment de définir des liens hypertexte.
Il possède trois marqueurs de language <html>, <head> et >body>.
Les développeurs web encourage a utiliser html5. Le W3C recommande le html5.
Avec html5 de nombreuses nouvelles balises sont arrivéeset beaucoup d'entre elles sont des balises dites sémantique car
elles permettent à la fois de servir de conteneur, mais aussi d'apporter des informations sur ce qu'elles contiennent.

Le markdown est une syntaxe légère, facile a apprendre, qui permet de créer les documents textes pour ces cours, mais aussi des cartes mentales, des diaporamas, des sites web, des flashs cards...

  1. Chaque site web commence par:!DOCTYPE html; indication qui permet d'indiquer au navigateur que cette page web est écrite en language html récent balise html, rôle différent: noter le titre ou le corps de texte.
  2. informations entre balise début, renseignée par deux crochets et balise fin, renseignée par une barre oblique entre deux crochets
  3. balise head: contient des informations qui renseigne le navigateur
  4. titre : balise title
  5. corps du texte : balise body
  6. 6 balises pour les titres : de h1 à h6, du plus important au moins important
  7. Paragraphe : balise p
  8. saut de ligne: balise br, pas de balise de fin!!!

Le Css signifie language de programmation. Il signification Cascading Style Sheets. Il permet de créer des sites web à l'apparence soignée comme celui-ci.
Dans CSS, les selecteurs sont utilisés pour cibler les éléments HTML à mettre en forme dans nos pages web.
CSS propose une grande diversité de selecteurs offrant ainsi une précision très fine dans la façon de cibler les éléments.

  1. Le selecteur css style: il permet de changer la couleur d'un mot ou d'une phrase, de rajouter un encadré ou une couleur de fond.
  2. Le selecteur par indentifiants: si on ne veut pas appliquer la fonction style a tout le paragraphe, il faut ajouter un selecteur par identifiant.
  3. Le selecteur classe: si on veut colorer le texte et ajouter une couleur de fond a un paragraphe, on ne peut pas ajouter deux balise par identifiant il faut donc ajouter le selecteur classe.

Toute ces infos sont entre accolades.
Le CSS est recommandé par le W3C.
